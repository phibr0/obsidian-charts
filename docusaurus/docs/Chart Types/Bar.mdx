# Bar Chart

## Example

````yaml
```chart
    type: bar
    labels: [Monday,Tuesday,Wednesday,Thursday,Friday, Saturday, Sunday, "next Week", "next Month"]
    series:
        - title: Title 1
        data: [1,2,3,4,5,6,7,8,9]
        - title: Title 2
        data: [5,4,3,2,1,0,-1,-2,-3]
```
````
